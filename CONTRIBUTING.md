# Contributing

Thanks for improving this Roblox Claude Skill.

## Contribution goals

Good contributions make Claude more useful for real Roblox production work:

- fewer vague answers;
- safer multiplayer code;
- clearer file placement instructions;
- better Studio test steps;
- stronger Rojo workflows;
- practical system blueprints;
- no fake Roblox APIs.

## Rules for changes

1. Keep `SKILL.md` concise. Put details in `reference/`, `systems/`, `playbooks/`, `tooling/`, `templates/`, or `examples/`.
2. Prefer practical checklists, file paths, and code templates over generic advice.
3. Do not add instructions that encourage trusting client data for currency, combat, rewards, saves, inventory, or quest progress.
4. Do not invent Roblox APIs. When an API is uncertain, instruct Claude to check Creator Hub or ask for project context.
5. Make templates beginner-readable but production-shaped.
6. Include test steps for new playbooks or system guides.

## Suggested PR format

- What workflow does this improve?
- Which files changed?
- How should Claude behave differently?
- What Roblox Studio test case proves the improvement?
