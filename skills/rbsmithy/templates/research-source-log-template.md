# Research Source Log

Use this file to track external sources used as inspiration or implementation references.

## Entry template

```text
Date checked:
Source title/name:
Source URL or asset ID:
Author/owner:
Source type: official docs / open-source repo / tutorial / Creator Store asset / user-owned file / public gameplay observation / other
License/terms:
Permission status: reusable / inspiration only / unknown / rejected
What was learned:
What was copied directly:
What was reimplemented originally:
Attribution required:
Files affected:
Risk level: low / medium / high
Notes:
```

## Rules

- If license is unknown, do not copy code.
- If source is a public game without source access, record it as gameplay observation only.
- If any code is reused under a license, also update `THIRD_PARTY_NOTICES.md`.
