# Roblox Part Model Factory

Use this when generating Studio-native models using Luau and Roblox Parts instead of external meshes.

## Best for

- blockouts;
- prototype ships;
- modular buildings;
- docks;
- props;
- debugging shapes;
- stylized low-poly assets;
- placeholder environment kits.

## Output rules

- create a top-level Model;
- use named helper functions;
- set Anchored intentionally;
- set CanCollide intentionally;
- use CollectionService tags when useful;
- group by purpose;
- add PrimaryPart if needed;
- avoid unnecessary physics;
- prefer simple shapes and strong silhouettes;
- include cleanup and placement instructions.
