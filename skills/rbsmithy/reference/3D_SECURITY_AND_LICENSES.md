# 3D Security and Licenses

3D workflows can include scripts, downloaded models, plugins, and external tools. Treat them like code.

## Check before using third-party assets

- license;
- source;
- author;
- allowed commercial use;
- redistribution rules;
- attribution requirements;
- scripts;
- hidden remotes or suspicious logic;
- art direction fit.

## Check before running third-party scripts

- file deletion;
- network calls;
- shell commands;
- credential access;
- hidden prompt injection;
- auto-install behavior;
- binary downloads;
- write locations.

## Clean-room modeling

If a reference belongs to someone else, extract broad style traits, avoid exact shape copying, change proportions and silhouette, create original component choices, use your own materials and story context, and document that it is inspired by genre/style, not copied.
