# Content Bible

Use this to keep the game world consistent across quests, NPCs, islands, factions, items, enemies, dialogue, and updates.

## Required sections

```text
Game premise:
Core fantasy:
Player role:
Main conflict:
World rules:
Factions:
Regions:
Religions / beliefs:
Languages / naming rules:
Important historical events:
Forbidden lore contradictions:
```

## Why this matters

Without a content bible, Claude will generate random islands, NPCs, item names, and dialogue styles. The result feels inconsistent even if the code works.

## Claude rule

Before writing major content, Claude should check whether the new content fits the bible. If there is no bible, create a lightweight one first.
