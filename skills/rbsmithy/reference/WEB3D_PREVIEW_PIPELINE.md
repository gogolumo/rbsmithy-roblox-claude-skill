# Web3D / Three.js Preview Pipeline

Use this when the user wants a browser preview or concept prototype before Roblox implementation.

## Purpose

A WebGL/Three.js preview can help with camera framing, silhouette, lighting mood, material direction, shader ideas, interactive asset previews, turntable previews, and portfolio-style presentation.

## Preview workflow

```text
Asset idea → Three.js preview → inspect silhouette/material/camera → simplify for Roblox → build/import in Studio
```

## Warning

Do not assume WebGL shaders directly work in Roblox. Translate intention, not exact implementation.
