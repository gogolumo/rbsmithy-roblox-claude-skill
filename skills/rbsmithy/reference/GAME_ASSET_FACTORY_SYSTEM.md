# Game Asset Factory System

Use this to produce consistent sets of assets rather than one-off models.

## Asset families

- dock kit;
- fisher village kit;
- pirate camp kit;
- ship upgrade kit;
- island rock kit;
- ruin kit;
- weapon kit;
- market stall kit;
- lighthouse kit;
- cave kit;
- underwater prop kit.

## Kit output

Every kit should include art direction, shared palette, modular pieces, hero piece, repeated props, collision rules, LOD plan, Roblox folders, naming conventions, and a test scene plan.
