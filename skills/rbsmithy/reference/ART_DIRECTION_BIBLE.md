# Art Direction Bible

Use this for visual consistency across UI, props, islands, NPCs, ships, effects, lighting, and Creator Store asset selection.

## Required sections

```text
Visual pillars:
Shape language:
Color language:
Material rules:
Architecture rules:
Faction clothing rules:
Prop rules:
UI style:
Lighting mood:
Forbidden styles:
Reference inspirations:
Performance budgets:
```

## Claude behavior

When generating or selecting assets, Claude should map the asset to the art bible:

```text
Does it match silhouette?
Does it match color/material rules?
Is it too detailed for the target platform?
Does it break the faction/region style?
Can it be simplified while keeping the idea?
```
