# Chat safety system blueprint

## Purpose
Support safe communication features without bypassing Roblox moderation.

## Rules
- Use Roblox text filtering for user-generated text.
- Prefer preset messages for gameplay coordination.
- Do not create unrestricted alternate chat systems.
- Add mute/block/report UI for social spaces.

## Examples
- crew ping wheel;
- emote wheel;
- preset voyage commands;
- filtered ship name.
