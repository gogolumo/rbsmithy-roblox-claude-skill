# Prompt: Find and Use Creator Store Assets

Use this prompt when asking Claude to help source Roblox Creator Store assets.

```text
Use the developing-roblox-games skill. I want to build an asset kit from Roblox Creator Store assets for this feature:

Feature: <describe the feature>
Game style: <low-poly / stylized / realistic / cozy / dark fantasy / etc.>
Needed assets: <list props, buildings, UI, audio, VFX, etc.>
Project mode: <Studio-only / Rojo / hybrid>
Access mode: <I will manually search Toolbox / Claude Code has Open Cloud access / make a placeholder manifest>
Performance target: <desktop only / mobile-friendly / low-end devices>
Scripts allowed from Store assets: <no / only after review>

Please produce:
1. search terms and filters;
2. selection criteria;
3. creator-store manifest template;
4. safe import workflow;
5. Rojo/Studio placement;
6. post-import cleanup checklist;
7. fallback procedural placeholders if assets are unavailable.
```
