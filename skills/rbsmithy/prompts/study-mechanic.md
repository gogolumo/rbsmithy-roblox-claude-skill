# Prompt: Study and Adapt a Mechanic

Use this prompt with the skill when you want Claude to learn from a public reference but build an original Roblox implementation.

```text
Use the developing-roblox-games skill.

I want to study this mechanic and build an original version for my Roblox game:

Reference source:
[link / video / screenshots / description / repository]

Mechanic:
[e.g. melee combat, sprint, dash, ship steering, anti-cheat, quest UI, dialogue camera]

My current project mode:
[Studio-only / Rojo / hybrid / not sure]

My current project files or tree:
[paste tree or attach files if available]

Rules:
- Do not copy protected code or assets.
- Use the reference only to understand the mechanic unless the source is clearly licensed for reuse.
- Create an original project-specific implementation.
- Use server-authoritative Roblox architecture.
- Include file paths, code, Studio placement, Remote contracts, test steps, and exploit checks.
```
