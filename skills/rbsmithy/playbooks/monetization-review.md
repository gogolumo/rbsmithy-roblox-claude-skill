# Monetization review playbook

Use before adding paid products.

## Review questions
- Does this product create unfair advantage?
- Does the player understand what they buy?
- Is the reward granted server-side?
- Is receipt processing idempotent?
- Is the base game intentionally painful without paying?
- What analytics prove the product is healthy?

## Output
- fairness rating;
- risks;
- safer alternatives;
- implementation checklist;
- test steps.
