# MicroProfiler review playbook

Use when FPS, frame spikes, physics, scripts, rendering, or UI are slow.

## Ask for
- client or server issue;
- device;
- Studio or published server;
- MicroProfiler screenshot/dump;
- Performance Summary FPS;
- reproduction steps.

## Review categories
- scripts;
- physics;
- rendering;
- UI;
- animation;
- networking;
- garbage collection.

## Output
- suspected hot spots;
- measurement plan;
- low-risk fixes first;
- deeper refactor if needed.
