# Funnel analysis playbook

Use when onboarding, tutorial, shop, quest, or progression metrics are weak.

## Steps
1. Identify the funnel goal.
2. List ordered player steps.
3. Add analytics events for each step.
4. Segment by device, new/returning player, and major content path.
5. Find largest drop-off.
6. Propose one focused fix.
7. Define success metric before implementation.

## Output
- funnel name;
- steps;
- event names;
- suspected drop-offs;
- experiment/fix idea;
- test plan.
