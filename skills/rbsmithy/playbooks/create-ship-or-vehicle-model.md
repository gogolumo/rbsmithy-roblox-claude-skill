# Create Ship or Vehicle Model

Use for ships, boats, carts, mounts, and other vehicles.

## Required planning

- hull silhouette;
- cockpit/helm/player position;
- collision hull;
- visual hull;
- physics root;
- decorative pieces;
- mast/sails/ropes;
- modular damage/upgrade pieces;
- LOD plan;
- mobile budget.

## Roblox ship rule

Separate visual model from physics/collision model whenever possible.
