# Generate Blender Python Asset

Use for procedural Blender scripts.

## Requirements

- start with config;
- name objects;
- generate clean geometry;
- keep material slots simple;
- avoid huge geometry on first run;
- include comments;
- include scale/origin notes;
- include Roblox import notes.

## After script

Always add how to run in Blender, what to inspect, what to export, what to fix manually, and Roblox import steps.
