# Create Roblox-Ready 3D Model

## Process

1. Define asset purpose.
2. Define style and silhouette.
3. Decide route: Blender manual, Blender Python, Roblox Part factory, or hybrid.
4. Set budget.
5. Plan components.
6. Plan collision.
7. Plan material simplification.
8. Plan export/import.
9. Plan QA.
