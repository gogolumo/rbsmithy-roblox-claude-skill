# UI device test playbook

Use before shipping HUDs, menus, shops, dialogue, or mobile controls.

## Test devices/modes
- 16:9 desktop;
- small mobile;
- tablet;
- gamepad;
- ultrawide;
- Roblox topbar visible;
- chat/playerlist enabled;
- different text sizes/localization lengths.

## Output
- device matrix;
- screenshots needed;
- layout risks;
- fixes.
