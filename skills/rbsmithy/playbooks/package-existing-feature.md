# Package existing feature playbook

Use when turning messy feature code into a reusable package.

## Steps
1. Identify public API.
2. Extract config.
3. Split server/client/shared.
4. Move constants into one module.
5. Document remotes.
6. Add analytics hooks.
7. Add README and TESTS.
8. Add migration guide.

## Output
- before/after tree;
- files to move;
- code patches;
- integration steps.
