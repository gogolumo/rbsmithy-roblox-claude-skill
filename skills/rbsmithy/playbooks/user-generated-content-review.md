# User-generated content review playbook

Use for ship names, guild names, signs, profiles, custom text, listings, or chat-like systems.

## Checks
- Does text come from a user?
- Is it filtered before display?
- Is it stored safely?
- Can players report it?
- Can users bypass chat rules?
- Are moderation tools server-side?

## Output
- risk level;
- filtering plan;
- report plan;
- test steps;
- safer alternative if needed.
