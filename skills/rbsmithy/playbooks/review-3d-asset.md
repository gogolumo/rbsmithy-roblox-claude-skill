# Review 3D Asset

## Review categories

- silhouette;
- style fit;
- topology;
- normals;
- UVs;
- materials;
- scale/origin;
- collision;
- LOD;
- mobile performance;
- Roblox import safety;
- reuse potential.

## Verdict

Use `GO`, `FIX FIRST`, or `REBUILD`.

Never say a model is production-ready without listing test steps.
