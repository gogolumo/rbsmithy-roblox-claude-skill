# Build Modular Environment Kit

Use for docks, villages, ruins, castles, caves, islands, markets, and ports.

## Output

- kit theme;
- repeated modules;
- hero pieces;
- connector pieces;
- scale grid;
- snap rules;
- materials;
- LOD/collision;
- Roblox folder structure;
- test scene plan.

## Rule

A kit is better than one giant model because it can be reused, optimized, streamed, and modified.
