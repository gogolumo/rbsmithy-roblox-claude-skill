# GitHub Workflow

## Recommended commands

```bash
git init
git add .
git commit -m "Initial Roblox project"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git push -u origin main
```

## Suggested checks

- Markdown files are readable.
- JSON project files parse.
- Luau files are formatted.
- Generated model blueprints are not huge binary blobs.
- `.rbxl` and `.rbxlx` place files are not committed unless intentionally versioned.
